# Быстрый старт

## Запуск приложения

### 1. Установка зависимостей

```bash
# В корне проекта
npm run install:all
```

Эта команда установит зависимости для:
- Корневого проекта (concurrently)
- Клиента (React, Vite, TailwindCSS)
- Сервера (Express, SQLite)

### 2. Запуск в режиме разработки

```bash
npm run dev
```

Эта команда запустит одновременно:
- Frontend на http://localhost:3000
- Backend API на http://localhost:5000

### 3. Открыть в браузере

Перейдите на http://localhost:3000 - вы увидите главную страницу с кнопками доступа ко всем интерфейсам.

## Интерфейсы

### Судейский интерфейс
- http://localhost:3000/judge/1 - Судья 1
- http://localhost:3000/judge/2 - Судья 2
- http://localhost:3000/judge/3 - Судья 3

Каждый судья может:
- Выбирать номинацию и команду
- Оценивать по 5 критериям от 0.1 до 10.0
- Оставлять комментарии к каждому критерию
- Видеть средний балл перед отправкой

### Админский интерфейс
- http://localhost:3000/admin

Администратор может:
- **Номинации**: добавлять и удалять номинации
- **Команды**: добавлять команды к номинациям
- **Текущая команда**: выбрать команду для зрительского голосования
- **Результаты**: просматривать финальную таблицу с баллами

### Зрительский интерфейс
- http://localhost:3000/spectator

Зрители могут:
- Видеть текущую команду (которую выбрал админ)
- Голосовать ползунком от 0.1 до 10.0
- Голосовать несколько раз

## Workflow использования

1. **Админ**: создаёт номинации (например: "Соло", "Группа", "Батл")
2. **Админ**: добавляет команды в каждую номинацию
3. **Судьи**: оценивают выступления команд по 5 критериям
4. **Админ**: переключает текущую команду для зрителей
5. **Зрители**: голосуют за текущую команду
6. **Админ**: смотрит результаты во вкладке "Результаты"

## Формула подсчета

Финальный балл рассчитывается так:

```
Судейский балл = среднее_взвешенное_по_всем_судьям
  где вес каждого критерия:
  - Техника: 1.2
  - Креативность: 1.0
  - Командная работа: 1.1
  - Презентация: 0.9
  - Общее впечатление: 1.3

Зрительский балл = средний_балл_зрителей × 0.5

ИТОГО = Судейский балл + Зрительский балл
```

## База данных

База данных SQLite автоматически создастся при первом запуске сервера:
- Путь: `database/championship.db`
- Таблицы: nominations, teams, scores, spectator_scores, current_team

## Troubleshooting

### Порт занят
Если порт 3000 или 5000 занят, измените в файлах:
- `client/vite.config.js` - порт клиента
- `server/index.js` - порт сервера

### Ошибка при установке
Убедитесь, что у вас установлены:
- Node.js >= 18
- npm >= 9

### База данных не создаётся
Проверьте права доступа к папке `database/`
