# Championship Scoring System

Система подсчета баллов для танцевальных чемпионатов с тремя интерфейсами:

## Интерфейсы

### 1. Судейский интерфейс
- 3 независимых судьи
- Выбор команды и номинации
- Оценка по 5 критериям (ползунки + комментарии)
- Все результаты сохраняются в БД

### 2. Админский интерфейс
- Добавление команд и номинаций
- Просмотр финальной таблицы с результатами
- Редактирование баллов
- Управление данными

### 3. Зрительский интерфейс
- Упрощенный интерфейс
- Один ползунок с общей оценкой
- Текущую команду переключает админ
- Отдельный подсчет зрительских голосов

## Технологии

- Frontend: React + Vite + TailwindCSS
- Backend: Node.js + Express
- База данных: SQLite
- Без авторизации

## Установка и запуск

### Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone <your-repo-url>
cd championship-scoring

# 2. Установить зависимости для всех частей проекта
npm run install:all

# 3. Запустить клиент и сервер одновременно
npm run dev
```

Приложение откроется на `http://localhost:3000`

### Отдельный запуск компонентов

```bash
# Только клиент (React)
npm run dev:client

# Только сервер (API)
npm run dev:server
```

### Сборка для продакшена

```bash
# Собрать фронтенд
npm run build

# Запустить сервер
npm start
```

## Формула подсчета баллов

Система использует взвешенные коэффициенты для расчета финального результата:

**Коэффициенты критериев:**
- Техника: 1.2
- Креативность: 1.0
- Командная работа: 1.1
- Презентация: 0.9
- Общее впечатление: 1.3
- Зрительские голоса: 0.5

**Расчет:**
1. Для каждого судьи: `(техника×1.2 + креативность×1.0 + командная_работа×1.1 + презентация×0.9 + общее×1.3) / сумма_коэффициентов`
2. Средний балл судей: `сумма_взвешенных_баллов / количество_судей`
3. Зрительский балл: `средний_балл_зрителей × 0.5`
4. **Финальный балл**: `балл_судей + зрительский_балл`
